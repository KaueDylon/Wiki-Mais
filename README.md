
# 📌 Wiki-Mais (W.I.P)

> Gerenciamento e criação de **Wikis**.

---

## 📚 Sobre o Projeto
 
Um gerenciador de wikis e artigos em Java, feito com o intuito de praticar o uso da Programação Orientada à Objetos e Banco de Dados com JDBC.

---


### Tecnologias Usadas:

- PostgreSQL 🐘
- Java 11 ☕
